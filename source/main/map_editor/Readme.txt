Project StuntPort
=================

Port of track editor from stuntrally (http://stuntrally.tuxfamily.org/).
Stuntrally uses OGRE 1.9 (+ Ogre::FrameListener) & latest MyGUI
    -> perfect fit for porting (matching license: GPLv3)
    
See files 'StuntRally_Readme.txt' and 'StuntRally_License.txt' for more info.
