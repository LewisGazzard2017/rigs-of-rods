 Rigs of Rods windows installer
 ------------------------------

 Required tools:
     * InnoSetup (http://www.jrsoftware.org/isinfo.php)
     * InnoTools Downloader (http://www.sherlocksoftware.org/page.php?id=50)

 Dependencies (put them to directory 'thirdparty')
     * Microsoft DirectX web installer
       (https://www.microsoft.com/en-us/download/details.aspx?id=35)
     * Microsoft Visual Studio 2015 Redistributable package
       (https://www.microsoft.com/en-us/download/details.aspx?id=48145)

 Instructions:
     Doubleclick 'rigs-of-rods.iss' to open it in InnoSetup IDE
     Edit constants on the top of the file as necessary
     Build the installer
